# Our Commitment
# We take security seriously at Olocus. As a privacy-first platform,
# protecting user data is our highest priority. We welcome security
# researchers to responsibly disclose vulnerabilities.

# Scope
# - olocus.com and all subdomains
# - Olocus mobile applications (iOS/Android)
# - Olocus API endpoints

# Safe Harbor
# We consider security research conducted in accordance with this policy
# as authorized conduct and will not pursue legal action.

# - Medium: 7 business days
# - Low: 14 business days
