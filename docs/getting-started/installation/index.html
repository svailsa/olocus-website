Installation Guide

Get Olocus Protocol up and running on your system in minutes.

System Requirementsâ€‹

Minimum Requirementsâ€‹

  • Rust: 1.75 or higher
  • RAM: 512 MB
  • Disk: 100 MB

That's It! ğŸ‰â€‹

No C compilers, no system libraries, no complex dependencies. Pure Rust, zero external C dependencies.

Installation Methodsâ€‹

From Source (Recommended)â€‹

# Clone from Codeberg (primary)
git clone https://codeberg.org/olocus/protocol.git
cd protocol

# Or from GitHub mirror (deprecated)
git clone https://github.com/olocus/protocol.git
cd protocol

# Build everything (pure Rust!)
cargo build --release

# Run tests (1,400+ passing!)
cargo test

# Try the demo
cargo run --example protocol_demo

Using Cargoâ€‹

# Install core library
cargo add olocus-core

# Add extensions as needed
cargo add olocus-location
cargo add olocus-trust
cargo add olocus-privacy

Platform-Specific Instructionsâ€‹

macOSâ€‹

# Install Rust if needed
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# Clone and build
git clone https://codeberg.org/olocus/protocol.git
cd protocol
cargo build --release

Linuxâ€‹

# Install Rust if needed
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# Clone and build
git clone https://codeberg.org/olocus/protocol.git
cd protocol
cargo build --release

Windowsâ€‹

# Install Rust from https://rustup.rs
# Then in PowerShell:
git clone https://codeberg.org/olocus/protocol.git
cd protocol
cargo build --release

Verify Installationâ€‹

Run the basic verification:

# Check version
cargo run --example get_version

# Output should be:
# Olocus Protocol v1.19.0
# Wire Version: 0x0105

Build Optionsâ€‹

Minimal Build (Core Only)â€‹

# Just the core (500 lines!)
cargo build -p olocus-core --release

Standard Build (Core + Common Extensions)â€‹

# Core + Location + Trust + Privacy
cargo build --release -p olocus-core \
  -p olocus-location \
  -p olocus-trust \
  -p olocus-privacy

Full Build (Everything)â€‹

# All 23 extensions
cargo build --all --release

Development Buildâ€‹

# Debug mode with symbols
cargo build

# With verbose output
cargo build -vv

# Run specific tests
cargo test -p olocus-core

Optional Featuresâ€‹

Some extensions have optional features:

[dependencies]
olocus-core = "0.1"
olocus-storage = { version = "0.1", features = ["rocksdb"] }
olocus-metrics = { version = "0.1", features = ["prometheus"] }

Docker Installationâ€‹

FROM rust:1.75-slim
WORKDIR /app
COPY . .
RUN cargo build --release
CMD ["./target/release/olocus-demo"]

Build and run:

docker build -t olocus .
docker run -it olocus

Troubleshootingâ€‹

Common Issuesâ€‹

Issue: cargo: command not found Solution: Install Rust from https://rustup.rs

Issue: Build fails with "cannot find -lssl" Solution: You don't need OpenSSL! Make sure you're building from the official repo.

Issue: Out of memory during build Solution: Use cargo build -j 1 to limit parallel jobs
